1)Для генерации документов и отчета нужно в shell прописать команду generate.
2)По умолчанию генерируется по 10 документов каждого типа. Чтобы изменить количество прописываем дополнительные параметры 
после команды generate.Пример: generate 11 12 13 сгенерирует 11 поручений, 12 входящих и 13 исходящих документов и отчет по ним.
