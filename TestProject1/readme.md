1)Для генерации документов и отчета нужно в shell прописать команду generate.
2)По умолчанию генерируется по 500 документов. Чтобы изменить количество прописываем дополнительные параметры 
после команды generate.Пример: generate --a 1000 сгенерирует 1000 документов и отчет по ним.
3)После в папке resources появятся json отчеты по каждому пользователю.
